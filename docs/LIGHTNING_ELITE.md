# ⚡ Lumaestro Lightning: Manual de Operações de Elite (V16)

O Lumaestro atingiu o ápice da autonomia industrial como um **Ecossistema Soberano**. Este manual cobre as funcionalidades de resiliência, comando direto e produção de inteligência (RLHF).

---

## 🛡️ 1. Manto de Resiliência (Self-Healing)
O enxame é imune a falhas de provedores individuais.
*   **Auto-Fallback**: Se o Gemini (Google) falhar, o Lumaestro alterna instantaneamente para **OpenAI** ou **Anthropic** (Claude) de forma silenciosa e transparente.
*   **Recuperação de Elite**: O roteador Go gerencia retentativas inteligentes para que o fluxo de trabalho nunca pare.

---

## 🕹️ 2. Console de Comando do Comandante (Vontade Direta)
Você não é apenas um observador; você é o Maestro.
*   Use o **Terminal de Comando** no Dashboard para enviar ordens diretas a qualquer agente.
*   **Intervenção Local**: Fale com o enxame para pedir relatórios, forçar mudanças de rumo ou ajustar o foco.

---

## 🚀 1. Motor APO & Beam Search
O enxame evolui através de 3 variantes estratégicas sempre que a performance cai. Você decide o futuro no **Córtex de Decisão**.

---

## 🛡️ 2. Filtro de Confiabilidade Gold
Novas variantes são testadas contra o "Passado Perfeito" antes de serem exibidas. Atente-se ao selo de **📦 GOLD Accuracy**:
*   🟢 **90-100%**: Estabilidade Industrial.
*   🔴 **Regressão**: O enxame detectou que a melhoria quebra o que já funcionava.

---

## 📂 3. Fábrica de Inteligência (RLHF)
Use o botão **"Gerar Dataset RLHF"** para exportar o conhecimento do enxame em formato `.jsonl`.
*   **Aplicações**: Treine seus próprios modelos (Fine-tuning) usando as "Gold Samples" e os caminhos aprovados pelo Comandante.
*   **Formato**: Padrão Conversacional (System/User/Assistant).

---

## 👁️ 4. Auditoria Multimodal (Visão)
O Dashboard agora monitora o que os agentes "veem".
*   Ao lado de cada rastro no **Fluxo de Consciência**, um ícone de **👁️** aparecerá se houver mídias (Imagens/PDFs) envolvidas.
*   Passe o mouse para auditar o caminho do arquivo visualizado.

---

## 🗄️ 5. DuckDB: A Consciência Colunar
Toda a telemetria é OTEL-compatible e persistida em `lumaestro_lightning.db`.
*   Para análise externa, use **"Exportar Traces"** (JSON).
